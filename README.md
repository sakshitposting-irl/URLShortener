# URLShortener

This is a URL Shortener built using Django, with authentication and authorization.
The URL Shortener is a web application that allows users to shorten URLs and access them later. This application supports multiple users, each with their own set of shortened URLs, and each user can only access their own shortened URLs.

Users with an authorized account can shorten URLs and access them later. The shortened URLs are unique to each user and are only accessible by the user who created them.

## Features
- User authentication and authorization
- Shorten URLs
- Access shortened URLs
- View all shortened URLs
- Delete shortened URLs

## Installation
1. Clone the repository
2. Install the required packages in the requirements.txt file


## Usage
1. Run the server using the command `python manage.py runserver`
